Dataframe Subset


The Subset utility allows you to easily generate subsets of pandas dataframes.

A Subset is a collection of rows in a dataframe selected via booean indexing.




You must import a dfc dataframe to proceed with the subset utility.


Status :

No dataframe imported to select for subsets


Get Dataframe Subset


Get Dataframe Subset


Select columns to keep or drop with the 'column_names_list.
Once you select the columns you can get a subset or define filters to get a subset.

Parameters:
*input_dataframe

The dfc dataframe to get a subset from.

*output_dataframe

The dfc dataframe to store subset in.

*column_names_list

The column names to retrieve from the input dataframe.

grab_columns_in_column_names_list_flag

Boolean flag to determine if keep or drop column_names_list columns.

csv_file_name

The csv file name to stor output_dataframe in. If blank don't save dataframe.

Commands: 
Define Filter(s)

Define filters to get the subset.

Get Subset

Use current filters to get the subset dataframe.

Scripting


The Scripting utility allows you to log dfc actions and review and or rerun them as a script.

Only actions that impact a dataframe are logged in the script file.



Turn On Scripting


Turning on scripting starts the logging of all commands.

Turn Off Scripting


Turning off scripting stops the logging of all commands.

Show Current Script


Displays the contents of the current script log.


Notes :

You can manually edit the script log as desired

The step numbers are commentary only and order is relative only.



Parameters:
*current_script

The current script in the loag that can be edited.

Commands: 
Run Current Script

Run the current script as displayed or changed.
