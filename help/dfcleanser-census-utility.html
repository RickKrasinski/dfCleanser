dfCleanser Census Utility


The Census utility allows you to retrieve census data and insert it into a pandas dataframe.

The census data included with this utility is derived from the US Census Bureau 2017 Data Release.


The Census data is broken into the following subsets for ease of use and can be downloaded and used independently of each section.

To review detailed census info for each subset click on the section titles below:

  • Economic
  • Education
  • Employment
  • Health Insurance
  • Housing
  • Immigration
  • Internet
  • Population
  • Social
  • Transportation

Data for each subset can be retrieved by the following:

  • City
  • County
  • State
  • Zip Code


Build Census Data

Select Census Data To Build


The Build Census Data functionality allows the user to decide what census data subsets to download and what summary types(zip code, ... state) should be built.

Once you select the census subsets and summary types click on the 'Build Selected Datasets' button to start the buuld process.

You can always come back and select additional subset to download at any time.

Download and Confirm Census Data


Once you select the subsets to download you must first download the subsets via following the links displayed and downloading the data to the defined location.

The path to download subsets to is a census working diretory (dfcleanser\files\census\working\) that is relative to the installation path of the dfcleanser package.

Due to size limitations of github you must download the individual subsets.


Once you download the requested subsets click on the 'Verify Census Zip Files Are Downloaded' button to verify and confirm that all rerquested subsets are local and available for processing.
