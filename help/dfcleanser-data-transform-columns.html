Add Column



Add column provides the capability to add a column to a pandas dataframe.


'Add New Column


Parameters:
* new_column_name

New added column name.

* new_column_name_df

Dataframe to add the column to.

* new_column_data_type

Datatype of the added column.

Commands: 
Get Column Values From File

Get the column values from a json list file.

Get Column Values From Code

Get column values from code.

Get Column Values From df

Get column values from a df column.

Add New Column From File


Add column from file provides the capability to add a column to a pandas dataframe from a file.

The file must be a json list with length of list equal to numer of rows in the dataframe.


'Add New Column From File



Parameters:
*new_column_name

New column name.

*new_column_name_df

Dataframe to add to.

*new_column_data_type

New Column datatype.

*column_values_file_name

New Column values file name.

*na_fill_value

Nan fill value. If None Nans inserted.

Results:       

A column is added from the values in the json list file.

Add Column From Code


Add column from user code or a dfc provided function.

Add Column From dfc Functions

Add column from dfc functions allows you to add a column using dfc functions to generate column values.


'Add New Column From dfc Functions


Select a dfc function to run to generate column values.


'Add New Column From dfc Functions


Once you select a dfc function to run the parms for that function are displayed and must be filled in.


Parameters:
*new_column_name

New column name.

*new_column_name_df

Dataframe to add to.

*new_column_data_type

New Column datatype. The datatype is initially determined by the dfc function selected.

*dfc_function

*dfc function selected to run.

dfc_function_code

dfc function call.

dfc_function_decription

dfc function description.

dftitle (function dependent)

df title to get column from. Input field dsiplayed based on dfc function.

dfcolname (function dependent)

df column name to apply funcrtion against. Input field dsiplayed based on dfc function.

User function code must be self contained so all objects must be imported.

All dfc functions generate a column values list equal to the number of rows in the df.

Results:       

The column values generated by the function are used in the added column.

Add Column From Dataframe


Add Column From Dataframe allows you to add a new column from other dataframe columns.


'Add New Column From df

Parameters:
*source_dataframe_title

Source dataframe title.

*source_column_name_to_retrieve

Source column name from source df.

*source_index_column_names

Source column names to use for matching output column names.

Selected column names are added to 'source_dataframe_index_list'.

*output_dataframe_title

Output dataframe title to write new column to.

*output_new_column_name

Output new column name.

*output_new_column_datatype

Output new column datatype. The datatype will default to the data type of the source column.

*output_index_column_names

Output columns to use for matching input column names.

Selected column names are added to 'output_dataframe_index_list'.

*source_dataframe_index_list

Source columns to use for matching output column names.

*output_dataframe_index_list

Output columns to use for matching input column names.

** dataframe_index_lists

The length of the source and output dataframe_index_list(s) must match.

When selecting values to add to new column the dataframe_index_list(s) are used as key fields to find values in the source df whose index values match the index values in the output df.

*na_fill_value

nan value to use when no value match between source and output. Default value is Nan.

Results:       

The matched list is added for the new column values.
