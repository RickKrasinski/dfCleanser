dataframe Manager


Pandas Dataframe Manager maintains a list of dfc dataframe objects that are then visible to all dfc functionality. A dfc dataframe is an object that has a title, pandas dataframe and user notes assigned to it. Any dataframe imported by dfc is automatically added to the dfc dataframe manager for dfc visibility. Any dataframe created outside dfc must be added to the dfc dataframe manager for dfc visibility.


Current dfCleanser dataframes


Input Fields:
*df_title

Select the dfc dataframe to work with.

df_num_rows

Number of rows in the currently selected df.

df_num_cols

Number of columns in the currently selected df.

dfc_notes

User notes assigned to the currently selected df.

Add df to dfc mgr


Add df to dfc dataframe Manager


Input Fields:
*df_title : string

The dfc title that is used to reference the dataframe within dfc.

df_name : pandas dataframe object

The pandas dataframe object to add to the dfc manager.

dfc_notes : string

User notes assigned to the dfc dataframe.
Results:

The dataframe is added to the dfc manager and made visible within dfc.

Drop df from dfc mgr


Input Fields:
*df_title : string

The dfc title that is used to reference the dataframe within dfc.
Results:

The dataframe is dropped from the dfc manager and no longer visible within dfc. Any dataframe imported by dfc is dropped from memory.

Update df in dfc mgr


Input Fields:
*df_title : string

The dfc title that is used to reference the dataframe within dfc.

dfc_notes : string

User notes assigned to the dfc dataframe.
Results:

The dfc manager dataframe notes are updated.
