Numeric Cols Row Search


The Numeric Cols Search allow you to retrieve the rows that match the numeric column values.


To save the df subset retrieved chnage the default df_output_title of 'temp_search_df'.
The 'temp_search_df' is dropped on every chapter reset.


Parameters:
*df_output_title

The title of the dataframe created by the rows matching the column values.

*column_names

df column names. For each column name selected a corrersponding column_values field is defined. Up to 3 column names can be selected.

*column_values

The column values that must match to get rows.

Commands: 
Get Rows Matching Numeric column_values

Build a df with rows matching the column values.

Get Non Numeric Columns

Display the search of Non Numeric Columns.

Examples:    
column_values

When you slect a column name the column_values field changes to reflect the column selected and the input goes active.

You can now enter the values to match in the column selected.

Non Numeric Cols Row Search


The Non Numeric Cols Search allow you to retrieve the rows that match the non numeric column values.



To save the df subset retrieved chnage the default df_output_title of 'temp_search_df'.
The 'temp_search_df' is dropped on every chapter reset.
