Data Import


Data Import is defined to provide importing of external datasources to a pandas dataframe.

Files



Pandas CSV Import


For input forms dfc keeps track of the previous input parms used and preloads the form with these values and does not display parms that were left blank.

If the user wants to see all parms then click on 'Show All Parameters' button to get a fuller set of parms as shown below.


Pandas CSV Import


For import forms the number of parms can be quite extensive allowing for customization of the import. If the user wants to add additional custom parms then enter additional parms as dict pairs with the import argument being the key and the parm(s) the value .


Pandas Dataframe Manager maintains a list of dfc dataframe objects that are then visible to all dfc functionality. A dfc dataframe is an object that has a title, pandas dataframe and user notes assigned to it.Any dataframe imported by dfc is automatically added to the dfc dataframe manager for dfc visibility. Any dataframe created outside dfc must be added to the dfc dataframe manager for dfc visibility.


CSV File


Pandas supports importing csv files via the pandas.read_csv() method. Click for details on the read_csv call.

FWF File


Pandas supports importing fwf files via the pandas.read_fwf() method. Click for details on the read_fwf call.

Excel File


Pandas supports importing excel files via the pandas.read_excel() method. Click for details on the read_excel call.

JSON File


Pandas supports importing json files via the pandas.read_json() method. Click for details on the read_json call.

HTML File


Pandas supports importing html files via the pandas.read_html() method. Click for details on the read_html call.


SQL


Pandas sql support allows for the importing of data from a sql database into a pandas dataframe.

The dataframe cleanser import automates connection to a common set of databases and corresponding python libs.

You must initially define the database, python lib and the connection parms.

SQL Libs


You can select which database and lib to connect to and then enter the appropriate connection parms.

The pandas import sql makes use of both a native and sqlalchemy connector libraries. The native library provides python connect functionality and the SQLAlchemy library provides a wrapper around the native library. Both are essential to using the pandas sql import functionality.

SQL Table


Pandas sql table support allows for the importing of data directly from a sql table into a pandas dataframe.


Pandas SQL Table Import - Postgresql

Input Fields:
*dataframe_title

The user defined df title that is visible in dfc.

*table_name

The database table to import.

*schema

The database schema to import.

*index_col

The table columns to use as index.

*coerce_float

Coerce floats from columns.

parse_dates
parse_dates_formats

'parse_dates' and 'parse_dates_formats' are merged into a dict before being passed to the read_sql_table api as the parse_dates arg.

If 'dates_format' is None the user must set the 'parse_date_formats' for each 'parse_dates' column.

If 'dates_format' is set the 'parse_dates_format' field is set to the 'dates_format' value.

*dates_format

Default date format to use on columns in the 'parse_dates' field.

*columns

Columns from the table to retrieve. If None all columns retrieved.

*chunk_size

Numer of rows to retrive at a time.

**

All other input parms are defined in the read_sql_table api.

Once you enter all the proper parms click on 'Import Table' to import the table as a dataframe.

Custom Import


dfc provides a generic way to import a dataframe and store it in the dfc manager for visibility in dfc.


Custom Import


Input Fields:
*dataframe_title

The user defined df title that is visible in dfc.

custom_import_code

The user can insert the proper code with which to import data into a dataframe. All external references need to be imported to become visible.

add_to_script_flag

A flag to determine if the custom inport call is stored into the notebook script log.
