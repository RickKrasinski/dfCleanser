The Common Data Structure Utility allows you to create lists or dicts hat can be used within the notebook.

You can retrieve a list by calling get_list('listname').

You can retrieve a dict by calling get_dict('dictname').





The lists and dicts contain both userr defined and system defined objects.

System objects are highilighted in yellow and can not be deleted.

The system objects are displayed in several chapters of the Dataframe Cleanser.
