Custom Export is defined to provide the ability to export data from a dataframe that is not supported in the standard exports.



The 'New Custom Export' button creates a python code cell for the user to test import code before including it in the 'Custom Export Code' form.

The 'Run Custom Export' button runs the code in the 'Custom Export Code' form and writes the notebook dataframe.

The 'Load New Custom Export' button loads the code in the python code cell in the 'Custom Export Code' form.

Only the code in the 'Custom Export Code' form is automatically loaded into the notebook scripting.
