Select/Change Geocoder provides the ability to select a supported geocoder.



For a general review of geocoding and initializing a geocoder please refer to the manual and review the '_init' function calls.

The specific parameter list for each of the geocoders supported can be found at :               

Any parameters created in the geoocoder _init calls are carried over to the 'geocode' and 'reverse' calls when appropriate. Any '_init' parameters can be overwritten in the 'geocode' and 'reverse' calls.
