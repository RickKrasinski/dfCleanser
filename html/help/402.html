Numeric Column cleansing alllows you to modify data content for the selected numeric column. To modify a specific column just select it from the 'Columns' table



The 'Column Stats' table shows info about the specific column.

The Skew and Kurtosis values are color coded to indicate level of normal distribution. Green indicates normal distribution, yellow questionable and red not normal distribution.

Along with the column stats data column cleansing provides additional details for unique values, graphs and outliers.
