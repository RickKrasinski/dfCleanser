The Datetime Transform Function allows you to work with datetime columns.



The 'Convert Column Datatype' button allows you change a column to a specific datetime datatype.

The 'Calculate timedelta Column' button allows you to calculate the timedelta between two columns.

The 'Split Column to date,time Columns' button allows you to split a datetime column to date and time columns.

The 'Merge Column from date,time Columns' button allows you to create a datetime column from separate date amd time columns.
