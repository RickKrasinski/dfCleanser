The dataframe concat functionality allows you to concatenate dataframes together.

The dataframe concatentation provided here is for simple concatenation of dataframes.

Simple concatenation does not support the multi index and labeling available for dataframe.

To use the full pandas concatentation click on Pandas help for details and do your own code cell.



The dataframes_to_concatenate defines a list of dataframe that are to be concatenated together in order of the list.

Each of the dataframes in the concat list must have been identified via calling dfcleanser.common.common_utils set_dc_dataframe("dfname").

The "default" dataframe points to the dataframe maintained by dfcleanser as you run dfcleanser.

The axis_to_concatenate_over defines whether to concatenate over rows or columns.

The join_value determines how to shape the concatenation. Refer to the pandas documentation for details.

The dataframe_to_store_results_in identifies the resultant dataframe that results are stored. Use set_dc_dataframe("dfname").

The reset_rows_index_flag determines whether to reset the rows index of the resultant dataframe.
