Row cleansing alllows you to modify data content for the selected row. Cleanse row provides a row browser to modify row values. You can select which row to update via Row search icon.



You can change a column value of 'Current Value' to the 'New Value' value by hitting the 'Change Value' key.

You can select 'New Value' and 'Current Value' vales by clicking on values in the 'Current Row' table. You can also enter 'New Value' manually

You can also drop the entire row by pressing the 'Drop Row' key.
