You can create user code from which the column values are generated.

The len of the new column values must match the len of the dataframe.



To generate a python code cell to test in hit the 'Open Test Code Cell' button.

To load the code from the test cell into the 'Add Column Code' input hit 'Get Code from Test Cell' button.

Only code in the 'Add Column Code' field can be run to add code for the column. The test cell is for testing only

You can save the 'Add Column Code' code as a generic function by pressing the 'Save Code as Generic' button.



Once you are happy with the 'Add Column Code' you can execute it to add a new column via pressing the 'Add New Column from Code' key.
