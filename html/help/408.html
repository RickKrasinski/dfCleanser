Once you select a non numeric column a 'Column Stats' table and a 'Unique Values and Counts' table is displayed.



The 'Unique Values and Counts' table lists the unique values for the specified column.



You can change all values equal to 'Current Value' to the 'New Value' value by hitting the 'Change Values' key.

You can select 'New Value' and 'Current Value' vales by clicking on values in the Unique Values table. You can also enter 'New Value' manually



You can additionally perform the functions below on the selected column.

The 'Drop Column' key al;lows you to drop the current column from the dataframe.

The 'Drop Current Vaalue Rows' key allows you to drop all rows whose column value equals the 'Current Value'.

The 'Drop Column Nan Rows' key allows you to drop all rows whose column value is Nan.

The 'Transform Column' key allows you to transform the column as defined in the Data Transform chapter.
