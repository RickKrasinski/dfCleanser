Generic Functions allows you to create a common function applied against the dataframe in the notebook.



The 'Open Test Code Cell' button allows you to open a python code cell for testing.

The 'Load Test Code' button loads the code from the test cell to the 'Function Code' field.

The 'Save Current Function' button allows you to save the function in the generic functions data store for later use.

The 'Delete Current Function' button allows you to delete the function from the generic functions data store.

The 'Run Function Code' button allows you to run the function against the dataframe amd store in the script log.
