Nans shows details on the number of Nans within the dataframe.



The 'Row Nan Stats' table shows the number of nans and number of rows with nans. You can also review the rows with most nans to selectively drop nans.

You can drop rows based on the % of cols that are nan or the column count that is nan. Just select the appropriate threshold and press the appropriate button





The 'Column Nan Stats' table shows the number of nans and number of columns with nans. You can also review the columns with most nans to selectively drop nans.

You can drop columns based on the % of rows that are nan or the row count that is nan. Just select the appropriate threshold and press the appropriate button
