Enter the list title in the 'List Name' field to store and retrieve by.

You can select a current list by selecting it from the 'Lists' table and it will be loaded into the 'List Values' field.



You can enter the list in the List Valuies box or cut and paste a list before saving it.

You can edit an existing list (system or user) and save it if desired.



Once you have created the proper list hit the 'Save To Notebook Lists' button.

You can delete the current list ('List Name' by hitting the 'Delete From Notebook Lists' button..

Certain lists(in yellow) are system lists and can not be deleted by you.
