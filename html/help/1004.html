The first parameters are required and are defined as follows :
  The 'query' parameter is used to define the address that coordinates are retrieved for. The more complete the address the faster the query.
  The 'number_of_results' parameter determines the max number of addresses retrieved to be displayed for the user to review.

Any parameters such as timeout that are defined in the gevoder setup are copied to the gewt coords automatically and can be overridden.
