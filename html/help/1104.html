The column_names_list field defines the list of columns to keep or drop for the subset.

The column name values can be selected by clicking on names in the 'Column Names' table or entered by hand (comma separated).



The keep_column_names_flag determines if the names in the column_names_list are to be kept in the new subset or dropped from the new subset.

The default value of True indicates to keep the column names and False indicates to drop the column names.

The csv_file_name parm allows the user to save the resultant subset in a csv file.



Once you select the parms you can add a filter for the subset or get the subset without filters.
