The 'from_column' and 'to_column' parameters define the address or coordinates columns to calculate distance between. For an address use a string column or build a list of column names to use ion oreder as the address string. For coordinates use column names containing a list or tuple of Latitude and Longitude. You can also define a list of lists for when latitude and longitude are in separate columns.
%nbsp;%nbsp;Latitude and Longitude in same column : '*_column' = column name%nbsp;%nbsp;Latitude and Longitude in unique columns : '*_column' = [latitude column name, longitude column name]

You must select a column name for the calculated distance to be stored in.

For the 'distance_algorithm' and 'elipsoid' parameters you can open the dict utility and review appropriate values.
