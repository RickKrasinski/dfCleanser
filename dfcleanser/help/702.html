The current script is displayed for the user to run.

The user is allowed to modify the script within the input form before doing any run or backup.



To run the current script hit the 'Run Current Script' button.

The script file is run internally and all changes are reflected in the final dataframe saved at the end of the script.

To restore the current script from the backup hit the 'Load Backup Script' button.

To store the current script as a backup hit the 'Backup Current Script' button.
