The dataframe schema functionality allows you to look ay the various column datatypes and changes them accordingly.



The datatype dropdow list shows the current datatype of the column and allows you to select a new datatype.

If the column contains nans then the user must choose a fillna value or select dropna.

If fillna is selected the user must define a value in the Nan Fill Value input.

If dropna is selectyed the 'Nan Fill Value' is disbaled and all nan rows are dropped.

Once all options are made hit the 'Change Datatype' button to change the datatype.
