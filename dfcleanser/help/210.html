Custom Import is defined to provide the ability to import data into a dataframe that is not supported in the standard imports.



The 'New Custom Code Cell' button creates a python code cell for the user to test import code before including it in the 'Custom Import Code' form.

The 'Load Custom Import from Code Cell' button copies the Custom Code Cell to the 'Custom Import Code' field. It also deletes the New code cell.

The 'Run Custom Import' button runs the code in the 'Custom Import Code' form and loads the notebook dataframe. After successfull run it is added to the scrpt.

The 'Add Custom Import Code To Script' button adds the 'Custom Import Code' field to the script.

Only the code in the 'Custom Import Code' form is automatically loaded into the notebook dataframe.
